1.启动 Redis
D:\.develop\Redis-x64-3.2.100\redis-server.exe D:\.develop\Redis-x64-3.2.100\redis.windows.conf
2.启动 Nacos (单机模式)
D:\.develop\nacos\nacos-server-2.2.3\bin\startup.cmd -m standalone
3.启动 minIO 服务
D:\.develop\minio\minio.exe server D:\.develop\minio\data

Redis -> Nacos -> minIO -> (等待10秒) -> Auth -> Gateway -> System -> Achievement ->  File

进入ui CMD,执行
(第一次) npm install --registry=https://registry.npmmirror.com
pnpm run dev

换电脑时，主要只要改一个文件就行：
dev-config.ps1
需要把里面这些路径改成新电脑的实际位置：
$ProjectRoot
$RedisExe
$RedisConfig
$NacosStartup
$MinioExe
$MinioDataDir
